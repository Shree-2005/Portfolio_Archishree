Duolingo Redesign

Refining mobile experience on iPhone 13

← Back

Overview

Modern, playful, and easier navigation to keep users engaged.

Design Process

Goal

To practice mobile UI design and understand how to create consistent layouts, hierarchy, and visual balance while working with an existing brand’s identity.

Approach

  • Visual Audit — Studied existing Duolingo screens and identified recurring elements.
  • Moodboard & Style Direction — Created a visual moodboard to guide color, spacing, and tone.
  • UI Kit & Components — Built a mini component system (buttons, cards, icons, progress bars).
  • High-Fidelity Screens — Designed 6 iPhone screens using the defined UI system.

Outcome

A clean and cohesive redesign of Duolingo’s mobile interface showcasing visual consistency, mobile layout design, and interaction detail skills.
