Spotify Redesign

Reimagining the desktop experience

← Back

Overview

A conceptual redesign of Spotify’s desktop interface created as a personal practice project to enhance my UI/UX design skills. The focus was on refining the visual design, maintaining consistency, and exploring new layout structures while preserving the essence of Spotify’s brand.

Design Process

Goal

To practice building a complete design system and improve my skills in typography, color, spacing, and interaction design by redesigning an existing product.

Approach

  • Visual Audit — Collected screenshots of the current interface to understand its structure and visual language.
  • Inspiration & Moodboard — Created a moodboard to define the new look and feel.
  • UI Style Guide — Defined colors, typography, spacing, grid, and reusable components.
  • High-Fidelity Screens — Designed 4 complete desktop screens following the new style guide.

Outcome

A visually clean and cohesive redesign of the Spotify desktop experience demonstrating visual design and layout skills.
